# 多入口项目，动态打包 demo

## multi-page

### devServer 启动，npm start +（项目名）

```
npm start all
```

or

```
npm start
```

or

```
npm start projectA  //启动 projectA
```

### webpack 打包， npm run build +（项目名）

```
npm run build all
```

or

```
npm run build
```

or

```
npm run build projectA  //启动 projectA
```
